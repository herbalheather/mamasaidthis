About Mama Said This

Hey, I'm Heather

I make free tools because I've been the one who needed them. From coding to herbalism, these tools are things I wish I could have found when I needed them most.

It's bad enough going through tough times without hitting paywalls for basic information. Mama said hell no to that.

My mom died when I was 19, just as I learned I was pregnant with my first. Food banks, emergency C-sections, multiple cash cars, and a pandemic with newborn twins taught me everything.

So I built this‚Äîpantry finders, budget help, resources that don't judge or gatekeep.

Why I Built This

I needed a pay stub generator. Every site wanted $20-40. I was broke. So I taught myself to code at 2am while my kids slept.

Then I kept building. Budget trackers. Grade calculators. Crisis resources. All the tools I wish existed when I was struggling.

Now I'm sharing them. Free. Forever. Because if you're like me, you're doing everything you can with what you have. You deserve tools that help, not gatekeep.

MY PROMISE:

  • NOT because I'm a tech genius.
  • NOT because I have funding.
  • NOT because I had time.
  • BECAUSE I've been there.
  • BECAUSE I know what it's like to need help.
  • BECAUSE I refuse to let others struggle when I can help.
